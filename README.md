AI Study Buddy is an interactive, GenAI-powered educational platform designed to transform static PDF documents into dynamic learning modules. Leveraging the **Gemini 3 Flash** model, the application automates the creation of summaries, quizzes, and flashcards to enhance student engagement and knowledge retention.

---

## üöÄ Key Features

* **Intelligent PDF Summarization:** Automatically parses dense academic documents and generates high-density, structured bullet points.
* **Dynamic MCQ Generation:** Transforms document context into Multiple Choice Questions with real-time scoring and answer validation.
* **Interactive Active Recall:** Utilizes custom CSS-animated 3D flashcards for effective technical term memorization.
* **Contextual AI Chat:** A dedicated interface for real-time doubt resolution and conceptual inquiries.
* **Automated Environment Setup:** One-click deployment script to handle dependency installation and server initialization.
